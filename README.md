# MysteryMail FORECASTs
*Facets of Reverse Engineering, Cryptography, and SkyTemple* (FORECAST) is a series of secretive hacking challenges that accompany MysteryMail (MM) events. Currently, there are two completed FORECASTs:
- Forbidden File (MM3)
- Deceptive Decoy (MM5)

This repository provides a technical analysis of each FORECAST, which includes solutions, behind-the-scenes details, as well as the source code when applicable. Most lore or implications gleaned from solving a FORECAST is, for the most part, out of the scope of this repository. ~~It'd also be unfair, since the owner of this repository is also the author of the FORECASTs.~~

Will contain spoilers for each event!
