# Deceptive Decoy - Source Code
Source code used to create the MM5 Top Menu! These files are intended to be used with [mm5-patches](https://github.com/Adex-8x/mm5-patches). Some files can simply be added to the repository, while others need to be outright replaced.

Quality not guaranteed!

- `src/decoi.c`
